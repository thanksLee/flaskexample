## 예제 만들하면서 얻은 내용
  
1.windows에서 python 개발

- python 설치 : 3.7
- pip install flask
- pip install virtualenv


2. 파이썬 가상환경
   virtualenv E:\myproject\MyPython

3. 가상환경으로 셋팅 변경
  - cd E:\myproject\MyPython
  - activate
  - deactivate

4. cd Scripts
- 앞으로 설치및 실행은 여기에 설치
- pip install flask
